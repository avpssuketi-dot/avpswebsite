\# Angels Valley Public School Website



Yeh school ke liye ek web portal hai jahan admissions ki inquiry aur notices manage kiye ja sakte hain.



\## 🚀 Features

\- \*\*Admission Inquiry Form:\*\* Parents ke liye admission form.

\- \*\*Notice Board:\*\* Admin panel ke zariye school ke notices dikhana.

\- \*\*PDF Generation:\*\* Inquiry details ko PDF mein download karne ki suvidha.

\- \*\*Responsive Design:\*\* Mobile aur Desktop dono par sahi dikhta hai.



\## 🛠 Tech Stack

\- \*\*Backend:\*\* Python, Flask

\- \*\*Database:\*\* SQLAlchemy (SQLite)

\- \*\*Frontend:\*\* HTML, CSS, JavaScript

\- \*\*Libraries:\*\* Pandas, ReportLab, Flask-Migrate
